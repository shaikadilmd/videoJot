## Built with
 - Node
 - Express
 - Mongoose
 - MongoDB
 - Express Handlebars
 - Passport for Local Authentication
 
## Usage
 Replace MONGO_DB_URI in app.js with your mongoDB URI.
```javascript
const uri = MONGO_DB_URI;
```

```
# Install dependencies
npm install

# Start 
npm start
```
